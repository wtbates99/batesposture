features

Everything you need, nothing you don't

A focused tool that stays out of your way until your posture needs attention.

score

Live 0-100 posture score

Seven weighted metrics — head tilt, neck angle, shoulder balance, spine alignment — combined into a single colour-coded score in your tray icon.

dashboard

Session dashboard

Sparkline history, session average, min/max, best streak, and duration — all in a floating window that remembers your history between sessions.

alerts

Smart notifications

Native desktop alerts with configurable threshold and cooldown. Focus mode silences reminders when you need to concentrate.

schedule

Interval scheduling

Track continuously or on a schedule. A built-in break reminder prompts you to stand up after 50 minutes of sitting.

logging

Local data logging

Optional SQLite logging of every session with CSV export. All data lives on your machine — nothing is sent anywhere.

performance

Adaptive performance

Automatically reduces camera resolution on low-end hardware. GPU acceleration toggle for faster MediaPipe inference on supported devices.

how it works

Simple by design

No complicated setup. Just your webcam and 30 seconds to calibrate.

1

Launch & calibrate

A short onboarding wizard captures a 6-second baseline of your natural posture so alerts are tuned to you, not a generic threshold.

2

Start tracking

Click "Start Tracking" from the tray icon. The icon turns green when posture is good and red when it degrades — no need to look at the app.

3

Receive alerts

When your score drops below the threshold, a native notification prompts you to sit up straight. A configurable cooldown prevents alert fatigue.

4

Review your session

Open the dashboard at any time to see your live camera feed, score history, streaks, and session stats.

run from source

Clone the repo, run with uv

No binaries, no installers. Requires Python 3.10+ and uv.

  1. 1Clone: git clone https://github.com/wtbates99/batesposture.git && cd batesposture
  2. 2Install dependencies: uv sync --all-groups
  3. 3Run: uv run batesposture  or  uv run python -m batesposture
  4. 4Grant camera permission when prompted, then complete the 6-second calibration on first launch.
  5. 5Linux / GNOME: may need the AppIndicator extension for tray support.

100% local processing

faq

Frequently asked questions

Quick answers about what BatesPosture is, how it works, and how to run it.

What is BatesPosture?

BatesPosture is a free, open-source desktop application that uses your webcam and Google MediaPipe to monitor and score your posture in real time. It produces a 0–100 posture score every second from seven weighted metrics — head tilt, neck angle, shoulder balance, spine alignment, and more — shown in a color-coded tray icon.

Is BatesPosture free?

Yes. BatesPosture is 100% free and open source. No subscriptions, no paid tiers, no premium features. Source on GitHub.

Does BatesPosture send my webcam video to the cloud?

No. All processing is local using MediaPipe. No video frames, pose data, or personal information ever leave your machine. Optional session logging is stored only in a local SQLite database.

Which operating systems are supported?

Windows 10/11 (64-bit), modern Linux (Ubuntu 20.04+, X11 or Wayland), and macOS. Requires Python 3.10+, the uv package manager, and a webcam.

How do I install BatesPosture?

Clone the repo: git clone https://github.com/wtbates99/batesposture.git, then uv sync --all-groups, then uv run batesposture. Grant camera permission and complete the 6-second calibration on first launch.

How is the posture score calculated?

MediaPipe pose landmarks feed seven weighted metrics — head tilt, neck angle, shoulder balance, spine alignment and others — combined into a single 0–100 score. A 6-second calibration tunes thresholds to your natural posture.

Will it slow down my computer?

No. Adaptive performance auto-reduces camera resolution on low-end hardware, and an optional GPU acceleration toggle speeds up MediaPipe inference on supported devices.

Can I export my posture data?

Yes. With optional logging enabled, sessions are written to a local SQLite database and can be exported to CSV. All data stays on your machine.
